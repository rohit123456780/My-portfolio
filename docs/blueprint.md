# **App Name**: CyberDeck: Mission Portfolio

## Core Features:

- Neural Command 3D Hero: An immersive Three.js-powered 3D hero section featuring a rotating security core that reacts to cursor movement and scroll depth.
- Mission Briefing CMS: A high-performance project and internship dashboard pulling dynamically from Firestore, featuring mission status tags and technical report summaries.
- AI Intelligence Analyst Tool: A feature that utilizes an LLM to automatically generate concise 'Intelligence Briefs' and technical narratives for each mission log based on user-provided raw project data.
- CertVault Security Hub: A filterable repository with search capabilities to manage and showcase 97+ certifications using specialized data nodes in Firestore.
- Secure Terminal HUD: A cinematic overlay including a global 'Command Palette' (Ctrl+K) for instant navigation across internships, skills, and publications.
- Authenticated Payload Console: An admin-exclusive portal using Firebase Auth for CRUD operations on work logs, projects, and site-wide metadata without code changes.
- Encrypted Transmission Channel: A contact form integrating Cloud Functions for server-side validation and automated logging of communication into Firestore.

## Style Guidelines:

- The palette is centered on a high-vibrancy 'Cyber Cyan' (#00C2FF) set against a deeply desaturated 'Dark Node' background (#0D1214) to emphasize a hacker aesthetic.
- An accent 'Bio-Toxic Mint' (#00FFAB) is used for active terminal text and 'Rank-Up' status indicators to provide high-contrast analogous depth.
- Headlines use 'Space Grotesk' for a computerized, technical feel; Body text utilizes 'Inter' for machine-like legibility; Terminal logs and status codes feature 'Source Code Pro'.
- Vector-based wireframe icons with thin 1px strokes, mimicking tactical HUD elements and security radar UI.
- A vertical 'Mission Timeline' layout utilizing cinematic parallax scroll-reveals and glassmorphism panel backgrounds with subtle chromatic aberration.
- Advanced GSAP scroll-triggering for 'Boot Sequence' intros and Framer Motion glitch-transitions when navigating between different site chapters.
